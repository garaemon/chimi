Chimi

What is Chimi?

ChimiはCommon Lispにおける, 雑多なユーティリティを 提供するパッケージです.

Chimiは ROLLプロジェクト の一環として開発されました.

Chimiは ROLL を構成する他のほとんどのパッケージで利用されています.

Chimiが提供するものは, 決して複雑なプログラムではありません. ですが, 有用なものでしょう.

もしもあなたが ROLL に興味をもったなら, まずChimiを理解するのが良いでしょう.

simple samples

using defclass* 

(require :chimi)
(use-package :chimi)
;; define <hoge> class                
(defclass* <hoge>
    ()
  ((a 100)
   (b nil)))
                
(defvar *hoge* (make-instance <hoge>))
(a-of *hoge*) ;; => 100
(setf (b-of *hoge*) 200) ;; => 200
                
(defvar *fuga* (make-instance <hoge> :a 200 :b "hogee"))
(a-of *fuga*) ;; => 200
(b-of *fuga*) ;; => "hogee"

using mtimer 

(defvar *timer* (make-mtimer))
                
(defun fib (n)
  (if (< n 2)
    n
    (+ (fib (- n 1)) (fib (- n 2))))))
         
(start-mtimer *timer*)
(fib 30)
(stop-mtimer *timer*) ;; -> foobar sec.. like  2.33

(plot-function "sin(x)" :dimension 2 :title "A")
(plot-function "cos(x)" :dimension 2 :clear nil :title "B")
